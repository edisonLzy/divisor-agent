Divisor Agent

A desktop-native AI agent runtime for Electron — bridging AI models with your local filesystem and terminal through a permission-controlled tool system.

✨ Core Features

Built for Power Users

Everything you need to run AI agents locally with full control

🤖

AI Agent Runtime

Powered by @mariozechner/pi-agent-core with multi-turn conversations, streaming responses, and real-time thinking process display.

🛠️

Local Tool System

Read/write files and execute terminal commands with dangerous command detection (e.g., rm -rf /). Direct filesystem access.

🔒

Permission Control

Real-time permission requests with "Remember this command" auto-approve. Session-scoped permissions keep your system safe.

🧩

Extension System

Dynamic extension discovery from ~/.pi/extensions/. Plugin slash commands and artifact panel for structured output.

💬

Rich Chat Interface

Virtualized message list, streaming Markdown with CJK/code/math/Mermaid, Shiki code highlighting with copy buttons.

📝

Rich Text Editor

TipTap editor with @ file search suggestions. Model selector grouped by provider. Permission mode selector built-in.
